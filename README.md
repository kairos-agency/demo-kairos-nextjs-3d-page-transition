# Next.js 3d page transition

## Setup

```shell script
pnpm install
# or
npm install
# or
yarn install
```

## Run development server

```shell script
pnpm dev
# or
npm run dev
# or
yarn run dev
```

## Build project

```shell script
pnpm build
# or
npm run build
# or
yarn run build
```
